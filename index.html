About

  • I am a PhD student currently awaiting my viva (exp. late 2025). My thesis is titled:

    On the Modelling of Temporal Data: Linear Logics in Event Logs and Deep Reinforcement Learning in Real-Time Simulations

    Bridging the gap between Temporal modelling as found in Business Process Management (BPMN) versus those in real-time systems exploited by (deep) reinforcement learning.
  • Prior to my PhD, and during my MSc in Computer Game Engineering, I was a junior network programmer at Lucid Games Ltd. developing Destruction AllStars, a new release for the (also newly released) PlayStation 5. Here, I became familiar with REST API technologies and Node.js. My significant contributions include:
    • Managing message protocols between the client and the web service; specifically, the integration of filtering into their matchmaking system
    • Extending Unreal's Gauntlet's Automation Framework. Here, I re-adapted the automation tests to communicate and synchronise with the web service, allowing for greater control from the network. I also wrote the HTTP request system for this project.
  • I have been programming since the start of my bachelor's degree in September 2017. Major milestones include:
    • PhD Thesis defence (exp. late 2025, Newcastle University)
    • Distinction, MSc Computer Game Engineering (Newcastle University)
    • 1:1, BSc Computer Science (Newcastle University)
  • My preferred languages are: Python; C#, C++, Java and C.
  • I have also produced some applications of my own. This includes my Pokémon analyser; which has also been used by other developers in the industry of Pokémon game balancing.
  • My GitHub, containing most of my projects, can be found here.
